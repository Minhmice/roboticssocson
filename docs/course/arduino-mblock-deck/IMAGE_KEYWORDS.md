## Asset checklist

- Prefer **WebP**, min **1200px** wide for hero slots, **4:3** or **16:9** matching layout
- Place files under `public/Images/Course/` or `public/Images/Mission/` then update `DECK_SLIDE_MEDIA` in `arduinoMblockDeckImages.ts`
- Run `npm run check:assets` after adding paths
- Raw sources available in `public/RAW/` (`arduino_board.jpg`, `arduino_logo.webp`, `scratch_logo.png`)
